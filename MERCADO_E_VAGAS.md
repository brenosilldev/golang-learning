## 🎯 Estratégia de Aplicação

### Fase 1 — Enquanto estuda (agora)

- [ ] Criar perfil em **Arc.dev**, **Turing** e **Lemon.io** (levam 2-4 semanas para aprovar)
- [ ] Otimizar **LinkedIn em inglês** — recrutadores buscam lá
- [ ] Assinar **HN Who's Hiring** — todo dia 1 do mês tem centenas de vagas

### Fase 2 — Ao terminar o curso Go (módulos 01-17)

- [ ] Aplicar em **golang.cafe** e **Remote OK** (filtro Golang)
- [ ] Aplicar em **We Work Remotely** e **Himalayas**
- [ ] Ter o **NexusMQ** no GitHub com README em inglês
- [ ] Meta: 5-10 aplicações por semana

### Fase 3 — Se for para Web3

- [ ] Aplicar em **crypto.jobs** e **web3.career**
- [ ] Ter o **ChainPulse** no GitHub
- [ ] Seguir protocolos no Twitter/X — muitas vagas Web3 são postadas lá
- [ ] Contribuir para projetos open source Web3 (go-ethereum, The Graph)

## 💡 Dica Final

> O **melhor dos dois mundos**: aprenda Go completo, pegue uma vaga backend remota ($4-7k/mês), e estude Web3 nas horas vagas. Quando estiver confiante, migre para Web3 ($7-12k/mês). Risco zero, crescimento garantido.

> **Meta realista**: aplicando em 5-10 vagas por semana, em 4-8 semanas de aplicação ativa, você deve conseguir pelo menos 3-5 entrevistas técnicas.
