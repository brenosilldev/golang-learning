## 🛠️ Ferramentas Essenciais

### Desenvolvimento Go

| Ferramenta | Para que serve |
|-----------|---------------|
| **VS Code + Go extension** | IDE mais popular para Go |
| **GoLand** (JetBrains) | IDE premium para Go (paga, mas excelente) |
| **golangci-lint** | Linter avançado (encontra bugs antes de rodar) |
| **Air** | Live reload para desenvolvimento web |
| **Delve** | Debugger para Go |
| **Go Swagger** | Gerar documentação de API automaticamente |

### Desenvolvimento Web3

| Ferramenta | Para que serve |
|-----------|---------------|
| **Ganache** | Blockchain Ethereum local |
| **Hardhat** | Framework de desenvolvimento Ethereum |
| **Foundry** | Framework em Rust (mais rápido que Hardhat) |
| **MetaMask** | Carteira browser para testar dApps |
| **Etherscan** | Block explorer (inspecionar contratos e TXs) |
| **Remix IDE** | IDE online para Solidity |
| **Alchemy** | Node-as-a-Service (conectar ao Ethereum sem rodar nó) |
| **Infura** | Alternativa ao Alchemy |
| **OpenZeppelin** | Contratos Solidity auditados e seguros |
| **The Graph** | Indexer descentralizado de blockchain |

## 🏆 Projetos Open Source para Estudar

### Go (código de referência)

| Projeto | O que é | O que aprender |
|---------|---------|---------------|
| **Caddy** | Web server | HTTP, middleware, plugins |
| **Hugo** | Gerador de sites | CLI, templates, performance |
| **Terraform** | Infraestrutura como código | Plugins, estado, providers |
| **Docker/Moby** | Containers | Networking, namespaces, cgroups |
| **CockroachDB** | Banco distribuído | Raft consensus, storage engine |
| **Prometheus** | Monitoramento | Métricas, PromQL, scraping |

### Web3 (código de referência)

| Projeto | O que é | O que aprender |
|---------|---------|---------------|
| **go-ethereum (geth)** | Client Ethereum oficial | EVM, P2P, consensus |
| **Prysm** | Client beacon chain (PoS) | Consensus, validator |
| **The Graph (graph-node)** | Indexer descentralizado | Subgraphs, indexing |
| **Uniswap v2/v3** | DEX (Solidity) | AMM, liquidity math |
| **OpenZeppelin** | Contratos padrão | Security patterns |
